# CLI Reference

`qpg` command groups:

- `init`
- `config`
- `source add|list|rm|rename`
- `context add|list|rm|generate`
- `auth check`
- `update`
- `status`
- `cleanup`
- `repair`
- `search`
- `vsearch`
- `query`
- `get`
- `schema`
- `mcp`

Use:

```bash
qpg --help
qpg <command> --help
qpg <command> <subcommand> --help
```

Common output/options:

- `--json`
- `--source`
- search/query-only: `--files`, `-n`, `--all`, `--min-score`, `--schema`, `--kind`
